## 🎬 How to Use

1. **Download** the HTML file from the link above
2. **Double-click** to open in Chrome, Firefox, Edge, or Safari
3. **Search stocks** using the top search bar
4. **Click any stock** in the watchlist to analyze it
5. **Toggle indicators** from the sidebar (SMA, RSI, MACD, Bollinger)
6. **Switch sections** via sidebar: Dashboard, Portfolio, Simulator, Watchlist
7. **Run simulations** in the Simulator tab to predict investment outcomes

The app simulates realistic stock price movements and AI predictions using mathematical models — it's designed to look and feel like a real trading platform!
